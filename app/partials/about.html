What is Meeting agenda builder?

It is a platform to create and plan your meetings in the easiest possible way.
You can add an activity of one of the four types Coffee Break, Group work,
Discussion and Presentation and also set the duration of each activity. You
can add days and then move your activities to any day.See the how many
activities you have through or activity color bar.It is a project work for the
course Interaction Programming and the Dynamic Web (DH2642)

  • Create Activity

    Add activity and its description

  • Set Type

    There are four types that youcan choose

  • Add to Day

    Create a day and drop your activities in it!

  • Visualization

    See time division of your activities by checking the color bar!
